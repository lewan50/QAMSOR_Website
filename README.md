Student Employability Study – UNOR

An interactive, quantitative comparative study evaluating predictive modeling approaches (linear and logistic regression) using mock interview competency scores for student employability analytics.

 📊 Project Overview
This web application visualizes analytics from the *Students' Employability Dataset – Philippines*. It evaluates four core competency dimensions:
- Mental Alertness
- Self-Confidence
- Ability to Present Ideas
- Communication Skills
